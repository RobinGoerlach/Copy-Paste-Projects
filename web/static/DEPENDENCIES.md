```markdown
# Project Dependencies

This file documents all external dependencies used in this project, including their purpose, version, and source.

## CSS Dependencies

1. **Normalize.css**
   - **Version:** 8.0.1
   - **Purpose:** Provides a consistent baseline for styles across browsers.
   - **File Location:** `assets/vendor/css/normalize.css`
   - **Source:** [https://necolas.github.io/normalize.css/](https://necolas.github.io/normalize.css/)

## JavaScript Dependencies

1. **jQuery**
   - **Version:** 3.6.0
   - **Purpose:** Simplifies DOM manipulation and event handling.
   - **File Location:** `assets/vendor/js/jquery.min.js`
   - **Source:** [https://jquery.com/](https://jquery.com/)

## Fonts

1. **Font Awesome**

   - **Version:** 5.15.4
   - **Purpose:** Provides scalable vector icons and social logos.
   - **File Location:** `assets/vendor/fonts/fontawesome.woff2`
   - **Source:** [https://fontawesome.com/](https://fontawesome.com/)

2. **Roboto**
   - **Version:** Latest
   - **Purpose:** Clean and modern sans-serif font.
   - **File Location:** `assets/fonts/roboto.woff2`
   - **Source:** [https://fonts.google.com/specimen/Roboto](https://fonts.google.com/specimen/Roboto)

## How to Update Dependencies

1. **Normalize.css:**
   - Visit the [Normalize.css GitHub repository](https://github.com/necolas/normalize.css) to download the latest version.
2. **jQuery:**
   - Download the latest version from [jQuery’s official website](https://jquery.com/download/).
3. **Font Awesome:**
   - Check for updates at [Font Awesome’s website](https://fontawesome.com/).
4. **Roboto Font:**
   - Download updates from [Google Fonts](https://fonts.google.com/).

## Notes

- All dependencies are stored in the `vendor/` folder to keep them separate from custom project files.
- Ensure to document the version and update sources when replacing or adding new dependencies.
```
